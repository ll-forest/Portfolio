Projects

Kyrrated: Personalized Time Management Web Application

View Application

The Challenge

In today's fast-paced world, effective time management is crucial, especially for individuals dealing with ADHD. Recognizing this need through a personal connection, I identified an opportunity to create a tailored solution that could make a real difference in someone's daily life.

The Solution

I developed Kyrrated, a specialized web application designed to assist individuals with ADHD in managing their time more effectively. The application features customizable schedules, reminders, and task prioritization tools, all presented in an intuitive, distraction-minimizing interface.

Technical Implementation

To ensure reliability and accessibility, I implemented a robust CI/CD pipeline using GitHub Actions for automatic deployment to AWS S3. This setup, combined with Route 53 for domain management, provides a seamless, always-up-to-date user experience. The project showcases my proficiency in cloud services, particularly AWS, and my ability to implement efficient DevOps practices.

The Learning Journey

This project deepened my understanding of user-centric design principles and the importance of accessibility in web applications. I gained valuable experience in creating responsive interfaces and implementing secure user authentication. Additionally, I enhanced my skills in AWS services, particularly in configuring S3 for web hosting and utilizing CloudFront for content delivery.

The Impact

Kyrrated has made a significant positive impact on its initial user, demonstrating the potential of technology to address specific personal challenges. This success has motivated me to continue refining the application, with plans to expand its feature set and potentially help a broader audience with similar needs.
